## Nota sobre Semillas:
Para mantener consistencia visual entre variantes del mismo personaje:
- Usa la **MISMA SEMILLA** para todas las plataformas de un personaje
- Varía solo el prompt según la plataforma
- Configura el **CFG Scale** entre 7-9 para mayor adherencia al prompt
- Usa **Steps** entre 30-50 para máximo detalle
- Formatos recomendados:
  - Instagram: 1024x1024 (1:1)
  - Facebook: 1080x1350 (4:5)
  - TikTok: 1080x1920 (9:16)
  - YouTube: 1920x1080 (16:9)
  - Discord: 1024x1024 (1:1, close-up crop)
